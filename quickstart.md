# 1. How to build from it

- Install base env as [joplin-doc](https://joplinapp.org/help/api/get_started/plugins), 
- Use `yo joplin` to make a base repo. 
- Install *todotxt-parser-js* by `npm install todotxt-parser-js`.
- Copy *src* folder
